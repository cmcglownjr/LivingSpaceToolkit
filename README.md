# Livingspace Toolkit

The purpose of this project is to create a toolkit that Livingspace LLC customer service can use to create custom orders
for Livingspace sunrooms.

## Getting Started

The requirements.txt file lists the required packages needed to run the gui application. There are also four custom
modules included.

### Prerequisits

The program needs the following packages: Pyside2, sys, os, math, and re.

### Installing

Use pyinstaller to generate the .exe file. Include the four modules, the .ui file, and the .qrc file.
Use --hidden-import PySide2.QtXml.

### Contributing
